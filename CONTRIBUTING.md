# How to Contribute

We are building this database together. Follow these steps to get credit for your work.

## 1. The Data Format
Copy this template for every new entry you add:

### [Title of Data Source]
* **ID:** `unique-id-here`
* **Provider:** [Name of Organization]
* **Update Frequency:** [e.g. Daily / Monthly / Annual]
* **Format:** `CSV` / `XLS` / `PDF` / `API`
* **Description:** A short sentence describing what is inside.
* **Direct Link:** [Download Data Here](url)
* **Added By:** [Your Name](Your Github link)  <-- DON'T FORGET THIS!
* **Stories in Action:**
    * [Headline of Story 1](url) - *Publication Name*
    * [Headline of Story 2](url) - *Publication Name*

## 2. How to add "Stories in Action"
When you add a dataset, you must find at least one news story that used this data.
1. Search for the dataset name (e.g., "IMF World Economic Outlook").
2. Find a high-quality article that cites the numbers.
3. Paste the link to show how professional journalists use this source.
4. If no story exists, write "None found yet."

## 3. How to Submit
1. Fork this repository.
2. Add your changes to the correct `.md` file.
3. Submit a Pull Request.
